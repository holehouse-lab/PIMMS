Similar to single_chain_polymer demo, this keyfile will work to simulate a single chain in isolation with the force field parameters for proteins.

The box dimensions for this system were chosen to be arbitrarily large.

For a sequence length of 500 amino acids the autocorrelation time for Rg was 1000 steps with these parameters. Therefore all the analyses are run every 1000 steps. In reality, this will be system dependent [length+chemistry], so use caution when performing analysis.
